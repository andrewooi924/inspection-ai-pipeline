# AI Inspection Intelligence Pipeline

A replayable staged AI pipeline for:

- inspection ingestion
- multilingual note normalisation
- deterministic defect prioritisation
- recurring issue detection
- explainable review generation
- routing and operational escalation

## Features

- Controlled vocabulary enforcement
- Multi-stage LLM orchestration
- Deterministic priority scoring
- Translation support
- Replayable execution
- Artifact persistence
- Validation framework
- LLM call logging
