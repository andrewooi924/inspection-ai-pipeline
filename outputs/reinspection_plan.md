# Reinspection Plan

## Priority Sampling
- Inspect recurring defect parts first.
- Prioritise stations involved in strong recurring patterns.

## Defect Signatures
- Solder void clustering near BGA and connector joints.
- Progressive scratch length increases.
- Pin alignment drift.

## Stop Criteria
- Stop production if multiple high severity recurring cases emerge in same station window.

## Continue Criteria
- Continue if reinspections remain below escalation thresholds.

- Quality
- Process Engineering
